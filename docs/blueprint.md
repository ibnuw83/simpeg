# **App Name**: Simpeg Smart

## Core Features:

- Pegawai Data Entry: Input form for employee master data including personal information, contact details, and initial employment information.
- Riwayat Jabatan Tracking: Log and display a history of an employee's positions within the organization, with start and end dates.
- Mutasi Management: Record and track employee transfers between departments or locations.
- Kenaikan Pangkat Automation: System to manage rank promotions with automated eligibility checks based on time in service and performance.
- Cuti Tracking: Manage and track employee leave requests, approvals, and balances.
- Document Management: Secure storage for important employee documents such as contracts, performance reviews, and certifications.
- AI-Powered Insights Tool: An AI tool that helps identify trends, such as which department has the highest turnover, to help the Dinas make more effective HR plans.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light blue-gray (#ECEFF1) for a clean, modern look.
- Accent color: Orange (#FF9800) to highlight important actions and notifications.
- Body and headline font: 'Inter', a sans-serif for readability and a modern feel.
- Use Material Design icons for a consistent and intuitive user experience.
- Clean and structured layout with clear navigation, ensuring ease of use.
- Subtle transitions and animations to provide feedback and improve user engagement.
